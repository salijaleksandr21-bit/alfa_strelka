## Юз-кейсы для конвертера валют
### Юз-кейс 1: Выбор валюты
* **ID:** ЮК-01
* **Название:** Выбор исходной и целевой валюты
* **Описание:** Пользователь выбирает исходную и целевую валюту из списка доступных валют.
* **Предусловие:** Пользователь находится на странице конвертера валют.
* **Основной поток:**
  1. Пользователь выбирает исходную валюту из выпадающего списка.
  2. Пользователь выбирает целевую валюту из выпадающего списка.
* **Постусловие:** Система отображает выбранные валюты.
* **Требование:** [БТ-01](#бизнес-требования-(бт))
* **Примечание:** Система должна поддерживать минимум 10 популярных валют, как указано в [БТ-05](#бизнес-требования-(бт)).

### Юз-кейс 2: Отображение курса обмена
* **ID:** ЮК-02
* **Название:** Отображение актуального курса обмена
* **Описание:** Система отображает актуальный курс обмена, полученный из внешнего API.
* **Предусловие:** Пользователь находится на странице конвертера валют.
* **Основной поток:**
  1. Система загружает актуальные курсы из API.
  2. Система отображает актуальный курс обмена.
* **Постусловие:** Пользователь видит актуальный курс обмена.
* **Требование:** [БТ-02](#бизнес-требования-(бт))

### Юз-кейс 3: Конвертация валюты
* **ID:** ЮК-03
* **Название:** Конвертация валюты
* **Описание:** Пользователь вводит сумму, и система мгновенно показывает результат конвертации.
* **Предусловие:** Пользователь находится на странице конвертера валют и выбрал исходную и целевую валюту.
* **Основной поток:**
  1. Пользователь вводит сумму.
  2. Система в реальном времени отображает сконвертированную сумму и курс.
* **Постусловие:** Пользователь видит результат конвертации.
* **Требование:** [БТ-03](#бизнес-требования-(бт))

### Юз-кейс 4: Смена направления
* **ID:** ЮК-04
* **Название:** Смена направления конвертации
* **Описание:** Пользователь нажимает кнопку «⇄», и исходная и целевая валюты меняются местами, результат пересчитывается.
* **Предусловие:** Пользователь находится на странице конвертера валют и выбрал исходную и целевую валюту.
* **Основной поток:**
  1. Пользователь нажимает кнопку «⇄».
  2. Система меняет местами исходную и целевую валюты.
  3. Система пересчитывает результат конвертации.
* **Постусловие:** Пользователь видит обновленный результат конвертации.
* **Требование:** [БТ-04](#бизнес-требования-(бт))

### Юз-кейс 5: Обработка недоступности API
* **ID:** ЮК-05
* **Название:** Обработка недоступности API
* **Описание:** Если API не отвечает, система использует последний закешированный курс.
* **Предусловие:** Пользователь находится на странице конвертера валют.
* **Основной поток:**
  1. API не отвечает.
  2. Система использует последний закешированный курс.
  3. Система отображает курс с пометкой о дате.
* **Постусловие:** Пользователь видит последний закешированный курс.
* **Требование:** [БТ-06](#бизнес-требования-(бт))

### Юз-кейс 6: Обработка некорректной суммы
* **ID:** ЮК-06
* **Название:** Обработка некорректной суммы
* **Описание:** Если пользователь вводит отрицательное число или нечисловое значение, система показывает сообщение «Введите корректную сумму» и не выполняет конвертацию.
* **Предусловие:** Пользователь находится на странице конвертера валют и выбрал исходную и целевую валюту.
* **Основной поток:**
  1. Пользователь вводит некорректную сумму.
  2. Система показывает сообщение «Введите корректную сумму».
  3. Система не выполняет конвертацию.
* **Постусловие:** Пользователь видит сообщение об ошибке.
* **Требование:** Некорректная сумма не должна приводить к выполнению конвертации.

### Юз-кейс 7: Доступность через веб-браузер
* **ID:** ЮК-07
* **Название:** Доступность через веб-браузер
* **Описание:** Пользователь может доступиться к конвертеру валют через веб-браузер.
* **Предусловие:** Пользователь имеет доступ к веб-браузеру.
* **Основной поток:**
  1. Пользователь открывает веб-браузер.
  2. Пользователь вводит адрес конвертера валют.
  3. Система отображает страницу конвертера валют.
* **Постусловие:** Пользователь может использовать конвертер валют.
* **Требование:** [БТ-07](#бизнес-требования-(бт))
