# Требования
Для запуска приложения необходимы следующие компоненты:
- Браузер с поддержкой HTML5, CSS3 и JavaScript (ECMAScript 6+)
- Подключение к Интернету для загрузки курсов валют

# Установка
Установка не требуется, так как приложение является веб-приложением и запускается直接 в браузере.

# Запуск приложения
1. Откройте веб-браузер и перейдите по адресу, где размещено приложение.
2. Выберите исходную и целевую валюты из выпадающих списков.
3. Введите сумму для конвертации.
4. Нажмите кнопку «Конвертировать», чтобы получить результат конвертации.

# Запуск тестов
Тесты не включены в данное приложение. Для проверки функциональности приложения необходимо вручную протестировать все возможные сценарии использования.

# Примечания по безопасности
- Приложение не отправляет данные наружу, все вводы санитизируются.
- Для локального использования держите браузер обновлённым.
- Приложение получает данные от стороннего API. Перед использованием в продакшене ознакомьтесь с политикой конфиденциальности этого API.
