# CurrFlow — Конвертер валют

CurrFlow — это легковесное веб-приложение для мгновенного пересчета валют с поддержкой темного интерфейса, автоматическим обновлением курсов и механизмом локального кеширования данных.

## Требования

Для запуска приложения не требуется установка сложных зависимостей или серверной среды. Необходим только современный веб-браузер:
- **Google Chrome** (последние 2 стабильные версии)
- **Mozilla Firefox** (последние 2 стабильные версии)
- **Safari** (последние 2 стабильные версии)
- **Microsoft Edge** (последние 2 стабильные версии)

## Установка

Приложение представляет собой статический веб-сайт, поэтому процесс установки минимален:

1. Склонируйте репозиторий или скачайте архив с исходным кодом.
2. Перейдите в директорию `src/`.
3. Убедитесь, что в папке находятся следующие файлы:
   - `index.html` — структура страницы.
   - `style.css` — стили оформления.
   - `app.js` — логика приложения.

## Запуск приложения

Вы можете запустить приложение двумя способами:

### Способ 1: Прямой запуск (простой)
Просто откройте файл `src/index.html` в любом современном браузере двойным кликом мыши.

### Способ 2: Через локальный сервер (рекомендуется)
Для корректной работы некоторых функций безопасности (например, Content Security Policy) рекомендуется использовать локальный сервер:
- Если у вас установлен **VS Code**, используйте расширение **Live Server**.
- Если установлен **Python**, выполните команду в папке `src/`:
  ```bash
  python -m http.server 8000
  ```
  После этого откройте в браузере адрес `http://localhost:8000`.

## Запуск тестов

Приложение написано на чистом JavaScript (Vanilla JS) и не содержит тяжелых фреймворков. Для проверки работоспособности выполните следующие ручные тесты:

1. **Проверка конвертации:** Введите число в поле «Сумма», выберите исходную и целевую валюты. Убедитесь, что результат обновляется мгновенно.
2. **Проверка валидации:** Попробуйте ввести буквы или отрицательные числа в поле суммы. Система должна блокировать ввод или выводить сообщение «Введите корректную сумму».
3. **Проверка смены валют:** Нажмите кнопку «⇄» и убедитесь, что валюты поменялись местами, а расчет произведен заново.
4. **Проверка кеширования:** Отключите интернет-соединение и обновите страницу. Приложение должно отобразить данные из кеша с уведомлением «Данные устарели».
5. **Проверка состояния:** Введите сумму и выберите валюты, затем обновите страницу. Приложение должно восстановить последние введенные данные из `localStorage`.

## Примечания по безопасности

Безопасность пользователя является приоритетом при разработке CurrFlow:

*   **Защита данных:** Приложение не отправляет ваши персональные данные или историю конвертаций на внешние серверы. Все расчеты и хранение состояния (`localStorage`) происходят локально в вашем браузере.
*   **Санитизация ввода:** Все вводимые данные проходят строгую фильтрацию. Использование `textContent` вместо `innerHTML` предотвращает возможность XSS-атак (межсайтового скриптинга).
*   **Безопасность API:** Приложение получает данные от стороннего API (`open.er-api.com`). Перед использованием приложения в продакшене ознакомьтесь с политикой конфиденциальности этого API.
*   **Политика CSP:** В приложении настроен заголовок `Content-Security-Policy`, который ограничивает источники загрузки скриптов и стилей, предотвращая выполнение несанкционированного кода.
*   **Рекомендация:** Для обеспечения максимальной безопасности держите ваш браузер обновлённым до последней версии.
