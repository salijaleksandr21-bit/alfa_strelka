### Резюме по безопасности:
1. **XSS Protection:** Все данные, извлекаемые из `localStorage` (ФТ-03) и ответы от API (ФТ-05, ФТ-06), проходят через процесс кодирования перед отображением в браузере.
2. **Input Validation:** Реализована строгая фильтрация ввода в поле суммы (ФТ-07, ФТ-08), исключающая возможность инъекций или ввода некорректных типов данных.
3. **API Security:** Введены требования по валидации схемы ответа API и ограничение запросов только доверенными эндпоинтами.
4. **Data Storage:** В `localStorage` сохраняются только нечувствительные данные (выбранные валюты, сумма), пароли и токены не используются и не хранятся.
