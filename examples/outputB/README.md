# МойТаскер — ToDo-доска задач

Простое одностраничное приложение для управления задачами с тремя статусами: «К выполнению», «В работе», «Выполнено».  
Работает полностью на стороне клиента, все данные сохраняются в `localStorage` вашего браузера.  

---

## Требования

- Современный браузер с поддержкой **ES6** и **Drag & Drop API** (Chrome, Firefox, Edge, Safari — последние версии).
- Включённый JavaScript.
- `localStorage` должен быть доступен (не заблокирован расширениями или настройками приватности).

---

## Установка

Установка не требуется. Достаточно скачать папку с файлами `index.html`, `style.css` и `app.js` и открыть `index.html` в браузере.

---

## Запуск приложения

1. **Способ 1 (простой):**  
   Дважды кликните по файлу `index.html` — страница откроется в браузере по умолчанию.

2. **Способ 2 (рекомендуемый, для корректной работы модулей):**  
   Запустите локальный веб-сервер, например:  
   ```bash
   cd /путь/к/папке/с/файлами
   npx serve .
   ```
   или с помощью Python:
   ```bash
   python3 -m http.server 8000
   ```
   Затем откройте `http://localhost:8000` в браузере.

   Использование локального сервера обеспечивает правильную обработку модулей JavaScript (`type="module"`) и избегает ограничений CORS, которые могут возникнуть при открытии файла напрямую.

## Примечания по безопасности

### Реализованные меры

- **Content Security Policy (CSP):**  
  В `<meta http-equiv="Content-Security-Policy">` заданы ограничения: загрузка ресурсов только с текущего домена, запрещён inline-скрипты (кроме `'unsafe-inline'` для совместимости, что допустимо для статического приложения). Политику можно ужесточить при развёртывании.

- **Санитизация пользовательского ввода:**  
  Все строки (название, описание) проходят очистку от опасных HTML-символов (`&`, `<`, `>`, `"`, `'`) перед вставкой в DOM. Для вывода используется свойство `textContent`, что предотвращает XSS-атаки.

- **Валидация входных данных:**  
  - Обязательное поле «Название» проверяется на пустоту.  
  - Длина строк ограничена: название — 500 символов, описание — 2000 символов.  
  - Статус задачи может быть только одним из трёх предопределённых значений (`todo`, `inprogress`, `done`).  
  - Фильтр задач принимает только значения из выпадающего списка.

- **Защита данных при загрузке из `localStorage`:**  
  При восстановлении задач проверяется, что данные являются массивом объектов с обязательными полями (`id`, `title`, `status`). Некорректные записи игнорируются.

- **Подтверждение необратимых действий:**  
  Удаление задачи возможно только после подтверждения через модальное окно, что предотвращает случайные потери данных.

- **Локальное хранение:**  
  Все данные сохраняются исключительно в `localStorage` браузера. Приложение **не отправляет данные на внешние серверы** и не использует сторонние API.

### Рекомендации

- **Держите браузер обновлённым** — это защитит от известных уязвимостей, связанных с JavaScript и DOM.
- **Не отключайте CSP** — политика безопасности установлена для минимизации рисков.
- **Очистка данных:** При необходимости удалите задачи через интерфейс или очистите `localStorage` вручную (`localStorage.removeItem('myTasker_tasks')`).
- **Конфиденциальность:** Имейте в виду, что данные хранятся в открытом виде (не зашифрованы). Не вводите чувствительную информацию (пароли, личные данные).
- **Использование с осторожностью:** Если вы планируете модифицировать код, убедитесь, что новые функции не нарушают санитизацию и валидацию. Любая уязвимость может быть эксплуадирована через другой браузерный контент на той же странице.
