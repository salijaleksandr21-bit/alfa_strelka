## Бизнес-требования (БТ)

| ID | Требование | Обязательность |
|----|-----------|---------------|
| БТ-01 | Пользователь должен иметь возможность выбрать исходную и целевую валюту из списка | Обязательное |
| БТ-02 | Система должна отображать актуальный курс обмена, полученный из внешнего API | Обязательное |
| БТ-03 | Пользователь вводит сумму — система мгновенно показывает результат конвертации | Обязательное |
| БТ-04 | Пользователь должен иметь возможность поменять валюты местами одной кнопкой | Обязательное |
| БТ-05 | Система должна поддерживать минимум 10 популярных валют (USD, EUR, RUB, GBP, JPY, CNY, TRY, KZT, BYN, GEL) | Обязательное |
| БТ-06 | При недоступности API система должна показывать последний закешированный курс с пометкой о дате | *Опциональное* |
| БТ-07 | Интерфейс должен быть доступен через веб-браузер | Обязательное |
