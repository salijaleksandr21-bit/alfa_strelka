# Бизнес-процесс (БП)

Название: Конвертация валюты  
Актор: Пользователь

## Основной поток
1. Пользователь открывает страницу конвертера.
2. Система загружает актуальные курсы из API и кеширует их.
3. Пользователь выбирает исходную валюту из выпадающего списка.
4. Пользователь выбирает целевую валюту.
5. Пользователь вводит сумму.
6. Система в реальном времени отображает сконвертированную сумму и курс.

## Альтернативный поток — смена направления
- Пользователь нажимает кнопку «⇄». Исходная и целевая валюты меняются местами, результат пересчитывается.

## Альтернативный поток — API недоступен
- 2a. Если API не отвечает, система использует последний закешированный курс. Рядом с курсом отображается: «Курс от [дата]. Актуальные данные временно недоступны».

## Альтернативный поток — некорректная сумма
- 5a. Если пользователь вводит отрицательное число или нечисловое значение, система показывает сообщение «Введите корректную сумму» и не выполняет конвертацию.
