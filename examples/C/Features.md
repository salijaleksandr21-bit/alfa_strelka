# Features – Конвертер валют с API

- Тёмная тема интерфейса
- Все надписи на русском языке
- Название приложения: «CurrFlow»
- Курсы обновляются каждые 60 секунд (автоматически)
- Список валют: дополнительно добавить UAH (украинская гривна), всего 11 валют (сверх БТ-05)
- При конвертации округление до 2 знаков после запятой
- Показывать не только результат, но и текущий курс (1 исходной валюты = X целевой)
- Индикатор загрузки при запросе к API (спиннер)
- При ошибке API показывать не только закешированный курс, но и жёлтую плашку «Данные устарели»
- Кнопка «⇄» меняет валюты местами и автоматически пересчитывает сумму (без дополнительного нажатия)
- Поле ввода суммы принимает только числа с точкой в качестве разделителя (валидация на лету)
- Поддержка клавиатуры: Enter выполняет конвертацию, если курс не изменился
- Сохранять последние выбранные валюты и сумму в localStorage, восстанавливать при следующем открытии страницы
