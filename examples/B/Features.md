# Features – Таск-трекер

- Светлая тема с акцентным цветом #4CAF50 (зелёный)
- Все надписи на русском языке
- Название приложения: «МойТаскер»
- При создании задачи поле «Описание» необязательное, но если заполнено – отображается под названием карточки
- Статусы задач отображаются в виде цветных меток: «К выполнению» – синий, «В работе» – жёлтый, «Выполнено» – зелёный
- Анимация при перетаскивании карточек (плавное перемещение)
- Подтверждение удаления через модальное окно (не через alert)
- Фильтрация по статусу – выпадающий список над колонками (реализация БТ-05)
- Данные хранятся в localStorage, автоматическое сохранение при каждом изменении (БТ-06)
- Возможность редактировать название и описание задачи по двойному клику на карточке
- Счётчик задач: отображается общее количество задач и сколько выполнено
