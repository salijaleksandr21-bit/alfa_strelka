
Бизнес-требования (БТ)

ID
Требование
Обязательность
БТ-01
Пользователь должен регистрироваться и входить в систему (логин/пароль). Данные пользователей хранятся в localStorage (имитация БД).
Обязательное
БТ-02
После входа пользователь видит список задач, назначенных на него, и может создавать новые задачи (название, описание, дедлайн, приоритет: низкий/средний/высокий).
Обязательное
БТ-03
Пользователь может назначить задачу другому зарегистрированному пользователю (роль "исполнитель").
Обязательное
БТ-04
Пользователь может изменить статус задачи: "Открыта", "В работе", "Выполнена".
Обязательное
БТ-05
Система отображает для каждой задачи иконку погоды (солнечно, облачно, дождь и т.п.) на основе текущей погоды в городе исполнителя (город указывается в профиле). Погода получается из внешнего бесплатного API (например, OpenWeatherMap или wttr.in).
Опциональное
БТ-06
Пользователь может видеть дашборд с метриками: количество задач по статусам, просроченные задачи (дедлайн прошёл, а задача не выполнена).
Обязательное
БТ-07
Все данные (пользователи, задачи, профили) должны сохраняться в localStorage и восстанавливаться после перезагрузки страницы.
Обязательное
БТ-08
Интерфейс адаптивен (работает на мобильных устройствах).
Опциональное
БТ-09
Система должна быть доступна через веб-браузер.
Обязательное
