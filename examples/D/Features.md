- Тёмная тема оформления
- Все надписи на русском языке
- Использовать API погоды: https://wttr.in/{город}?format=j1 (бесплатно, без ключа)
- Обновлять погоду при загрузке страницы и по кнопке «Обновить погоду»
- Хранить последние полученные данные о погоде в localStorage, чтобы при офлайне показывать кэш
- Дата и время в формате ДД.ММ.ГГГГ ЧЧ:ММ
- Приоритеты визуально выделять цветом: высокий — красный, средний — жёлтый, низкий — зелёный
