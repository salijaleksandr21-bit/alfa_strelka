# CurrFlow — Конвертер валют

CurrFlow — это клиентское веб-приложение для конвертации валют с тёмной темой интерфейса и автоматическим обновлением курсов.

## Требования

- Современный веб-браузер (Chrome 90+, Firefox 90+, Safari 15+, Edge 90+)
- Подключение к интернету для получения актуальных курсов валют
- Включённый JavaScript

## Установка

Установка не требуется. Приложение работает полностью на стороне клиента.

## Запуск приложения

1. Скачайте все файлы проекта в одну папку:
   - `index.html`
   - `style.css`
   - `app.js`

2. Откройте файл `index.html` в вашем браузере.

Приложение автоматически:
- Загрузит актуальные курсы валют с внешнего API
- Восстановит последние выбранные валюты и сумму из локального хранилища браузера
- Начнёт автоматическое обновление курсов каждые 60 секунд

## Использование

1. **Выбор валют**: Используйте выпадающие списки «Исходная валюта» и «Целевая валюта»
2. **Ввод суммы**: Введите сумму в поле «Сумма» (только цифры и десятичная точка)
3. **Смена валют местами**: Нажмите кнопку «⇄»
4. **Принудительная конвертация**: Нажмите Enter в поле ввода суммы

## Запуск тестов

Тесты не предусмотрены, так как приложение не имеет серверной части и не требует сборки.

## Примечания по безопасности

Приложение не отправляет данные наружу, все вводы санитизируются. Для локального использования держите браузер обновлённым.

**Важно**: Приложение получает данные от стороннего API (`https://api.exchangerate-api.com`). Перед использованием в продакшене ознакомьтесь с политикой конфиденциальности этого API.

### Дополнительные меры безопасности

- Все пользовательские вводы проходят строгую валидацию на стороне клиента
- Данные в локальном хранилище не содержат конфиденциальной информации
- Приложение использует Content Security Policy для защиты от XSS-атак
- Все вызовы API выполняются по протоколу HTTPS
- При восстановлении данных из localStorage проверяется их формат и допустимость значений
