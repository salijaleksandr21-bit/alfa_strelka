# Features – Веб-калькулятор

- Тёмная тема интерфейса по умолчанию
- При нажатии на кнопки должен звучать короткий клик (опционально, если успеем)
- Все надписи на русском языке
- Название приложения: «QuickCalc»
- Поле ввода показывает предыдущее выражение мелким шрифтом, текущий ввод — крупным
- При делении на ноль показывать не только текст, но и красную рамку вокруг поля ввода
- История вычислений сохраняется в localStorage и доступна после перезагрузки страницы (расширение БТ-02)
- Кнопка «C» сбрасывает только текущий ввод, история не очищается
- Для очистки истории – отдельная кнопка «Очистить историю»
- Адаптивный дизайн: корректно отображается на мобильных устройствах (ширина ≤ 600px)
