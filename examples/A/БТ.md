## Бизнес-требования (БТ)

| ID | Требование | Обязательность |
|----|-----------|---------------|
| БТ-01 | Пользователь должен иметь возможность создавать задачи с названием и описанием | Обязательное |
| БТ-02 | Каждая задача имеет один из трёх статусов: «К выполнению», «В работе», «Выполнено» | Обязательное |
| БТ-03 | Пользователь должен иметь возможность перемещать задачу между статусами | Обязательное |
| БТ-04 | Пользователь должен иметь возможность удалять задачи | Обязательное |
| БТ-05 | Пользователь должен иметь возможность фильтровать задачи по статусу | *Опциональное* |
| БТ-06 | Данные задач должны сохраняться между сессиями (локальное хранилище) | *Опциональное* |
| БТ-07 | Интерфейс должен быть доступен через веб-браузер | Обязательное |
