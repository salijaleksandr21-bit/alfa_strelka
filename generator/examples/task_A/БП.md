# Бизнес-процесс: Выполнение вычисления

Актор: Пользователь

Основной поток:
1. Пользователь открывает веб-страницу калькулятора.
2. Пользователь вводит первое число, нажимая на цифровые кнопки.
3. Пользователь выбирает операцию (+, −, ×, ÷).
4. Пользователь вводит второе число.
5. Пользователь нажимает «=».
6. Система отображает результат на экране.

Альтернативный поток — деление на ноль:
5a. Если второе число равно нулю и выбрана операция деления, система отображает сообщение «Ошибка: деление на ноль».

Альтернативный поток — сброс:
*. В любой момент пользователь нажимает «C» — поле ввода очищается.
