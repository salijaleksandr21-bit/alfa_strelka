# Бизнес-требования (БТ) для Веб-калькулятора

| ID | Требование | Обязательность |
|----|------------|----------------|
| БТ-01 | Пользователь должен иметь возможность выполнять базовые арифметические операции: сложение, вычитание, умножение, деление | Обязательное |
| БТ-02 | Пользователь должен видеть историю последних 10 вычислений в текущей сессии | Опциональное |
| БТ-03 | Пользователь должен иметь возможность очистить текущий ввод и полностью сбросить калькулятор | Обязательное |
| БТ-04 | Система должна корректно обрабатывать деление на ноль, показывая понятное сообщение об ошибке | Обязательное |
| БТ-05 | Интерфейс должен быть доступен через веб-браузер без установки дополнительного ПО | Обязательное |
