# CurrFlow - Конвертер валют

## Описание
**CurrFlow** — это безопасное веб-приложение для конвертации валют в реальном времени. Приложение разработано с использованием чистого HTML, CSS и JavaScript (Vanilla). Основной упор сделан на безопасность данных пользователя, защиту от XSS-атак и корректную работу с внешними источниками данных.

## Требования
Для корректной работы приложения необходимы следующие условия:
- **Браузер:** Современный веб-браузер (Chrome, Firefox, Safari, Edge) с поддержкой ES6 и HTTPS.
- **Интернет-соединение:** Требуется для получения актуальных курсов валют от внешнего API.
- **Хранилище:** Разрешено использование `localStorage` в браузере для сохранения настроек и кэширования данных.
- **Безопасность:** Рекомендуется использовать актуальную версию браузера для поддержки современных протоколов шифрования и политик безопасности (CSP).

## Установка
Установка программного обеспечения не требуется. Приложение является статическим веб-ресурсом.
1. Скачайте файлы проекта (`index.html`, `style.css`, `app.js`).
2. Разместите их в одной папке на вашем компьютере.
3. (Опционально) Для корректной работы заголовков безопасности рекомендуется запустить локальный сервер (например, через `python -m http.server` или расширение Live Server в VS Code), хотя приложение также может работать при прямом открытии файла.

## Запуск приложения
1. Откройте файл `index.html` в веб-браузере.
2. Выберите исходную и целевую валюты из выпадающих списков.
3. Введите сумму для конвертации.
4. Результат отобразится автоматически. Приложение также поддерживает клавишу `Enter` для подтверждения ввода.

## Запуск тестов
Автоматизированный набор тестов в текущей версии проекта не включен. Рекомендуется проводить ручное тестирование функционала:
- Проверка корректности конвертации для различных пар валют.
- Проверка работы кэширования (отключение интернета и повторная загрузка).
- Проверка валидации ввода (попытка ввода букв в поле суммы).
- Проверка отображения сообщений об ошибках при недоступности API.

## Примечания по безопасности

### Защита данных
**Приложение не отправляет данные наружу, все вводы санитизируются. Для локального использования держите браузер обновлённым.**

Приложение реализует следующие меры безопасности:
- **Content Security Policy (CSP):** В заголовках страницы настроена политика безопасности, запрещающая выполнение инлайн-скриптов и загрузку ресурсов с недоверенных доменов. Не рекомендуется отключать CSP.
- **Санитизация ввода:** Все данные, вводимые пользователем, очищаются от недопустимых символов перед обработкой.
- **Защита от XSS:** Вывод данных в DOM осуществляется через безопасные методы (`textContent`), а не через `innerHTML` для пользовательских данных.

### Внешние API
**Приложение получает данные от стороннего API. Перед использованием в продакшене ознакомьтесь с политикой конфиденциальности этого API.**

- Приложение использует публичный API для получения курсов валют (`https://api.exchangerate-api.com`).
- Запросы осуществляются только по защищенному протоколу HTTPS.
- Ключи доступа к API не хранятся в клиентском коде (в данной реализации используется публичный эндпоинт без ключей).
- Данные кэшируются локально в браузере пользователя и не передаются третьим лицам.

### Конфиденциальность
- В `localStorage` сохраняются только настройки интерфейса (выбранные валюты) и кэш курсов.
- Личные данные пользователя, пароли или токены авторизации не собираются и не хранятся.
- При сбое соединения приложение отображает общие сообщения об ошибках, не раскрывая внутреннюю структуру системы.
