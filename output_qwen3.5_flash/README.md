# CurrFlow — Конвертер валют

**CurrFlow** — это безопасное и легковесное веб-приложение для конвертации валют. Приложение работает в браузере, использует тёмную тему и обеспечивает защиту данных пользователя на уровне клиентского кода.

## Требования

Для корректной работы приложения необходимы следующие условия:

- **Браузер:** Современный браузер (Chrome, Firefox, Safari, Edge) с поддержкой ES6+ и Fetch API.
- **Сетевое подключение:** Требуется для получения актуальных курсов валют (при отсутствии локального кэша).
- **Сервер:** Не требуется. Приложение является статическим (HTML/CSS/JS). Рекомендуется запускать через локальный веб-сервер для корректной работы политик безопасности (CSP) и CORS.

## Установка

Приложение не требует сложной установки.

1.  Скачайте исходные файлы проекта:
    - `index.html`
    - `style.css`
    - `app.js`
2.  Разместите файлы в одной директории.

*Рекомендация:* Для запуска используйте локальный сервер (например, расширение Live Server в VS Code или `python -m http.server`), чтобы избежать ограничений протокола `file://`.

## Запуск приложения

1.  Запустите локальный веб-сервер в директории с файлами.
2.  Откройте в браузере адрес, указанный в терминале (обычно `http://localhost:8000` или `http://127.0.0.1:5500`).
3.  Интерфейс загрузится с тёмной темой и заголовком «CurrFlow».

## Запуск тестов

Автоматизированный набор тестов в проекте не предусмотрен. Для проверки работоспособности выполните следующие ручные проверки:

1.  **Валидация ввода:** Попробуйте ввести текст или отрицательные числа в поле «Сумма». Система должна отклонить ввод и показать сообщение об ошибке.
2.  **Конвертация:** Введите число и выберите валюты. Убедитесь, что результат пересчитывается мгновенно.
3.  **Смена валют:** Нажмите кнопку «⇄». Валюты должны поменяться местами, сумма пересчитаться.
4.  **Обработка ошибок API:** Отключите интернет или заблокируйте запрос к API. Убедитесь, что приложение использует кэшированные данные и показывает предупреждение «Данные устарели».
5.  **LocalStorage:** Обновите страницу. Проверьте, что выбранные валюты и сумма сохранились.

## Примечания по безопасности

Приложение разработано с соблюдением принципов **Security by Design**.

### Основные меры защиты
- **Content Security Policy (CSP):** В заголовках страницы настроена политика безопасности, запрещающая выполнение скриптов из недоверенных источников и использование опасных функций (например, `eval`).
- **Санитизация ввода:** Все пользовательские данные проверяются на валидность перед обработкой. Вывод данных в DOM осуществляется через `textContent`, что предотвращает XSS-атаки.
- **Локальное хранение:** Данные сохраняются только в `localStorage` браузера пользователя. Чувствительная информация (пароли, токены) не хранится.

### Важное предупреждение
> **Приложение не отправляет данные наружу, все вводы санитизируются. Для локального использования держите браузер обновлённым.**

### Внешние API
Приложение получает данные о курсах валют от стороннего сервиса.

> **Приложение получает данные от стороннего API. Перед использованием в продакшене ознакомьтесь с политикой конфиденциальности этого API.**

### Рекомендации по эксплуатации
- Не отключайте заголовки безопасности (CSP, X-Frame-Options) в настройках браузера или сервера.
- Избегайте использования приложения в публичных киосках без настройки очистки `localStorage` после сессии.
- При развертывании в продакшене убедитесь, что соединение с API происходит по протоколу HTTPS.
